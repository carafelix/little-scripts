# little-scripts
QoL or pure fun scripts

## Summary

- ```nilsimsaHash...etc.js``` inputs a text file > nilsimsa > char conversion from hext to letters > outputs a txt file.
- ```rename-manga.js``` checks for every .rar file inside the folder containing the script, if matches rename it to .cbz


## Do's and don'ts

- Be aware that any of this scripts checks for sensitive paths or file overwrite. Be careful.
